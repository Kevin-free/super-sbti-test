# SBTI 测试（重制游戏版）

基于 B站UP主 [蛆肉儿串儿](https://www.bilibili.com/video/BV1LpDHByET6/) 的原创构思，重制为一个更有趣、支持社交分享与解锁报告的 React 单页应用。

## 🌟 特性
- 现代化的 UI / UX (Tailwind CSS, Framer Motion)
- 响应式适配移动端
- 完整的 15 维度逻辑还原
- 支持前端模拟付费解锁功能（仅作体验或演示用）
- 一键生成结果分享海报 (html2canvas)

## 🚀 启动与开发

```bash
npm install
npm run dev
```

构建生产环境：
```bash
npm run build
```
